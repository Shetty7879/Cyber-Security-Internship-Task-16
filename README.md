# Cyber Security Internship – Task 16

## Task Title
Incident Response & Security Breach Simulation

## Objective
To understand the incident response lifecycle and simulate a basic security incident for analysis and response.

## Tools Used
- Windows Event Viewer / Linux Logs
- TheHive (Conceptual)

## Incident Simulated
- Multiple failed login attempts
- Suspicious authentication activity
- Possible brute-force attack scenario

## Incident Response Phases
- Detection and analysis
- Incident classification
- Containment
- Eradication
- Recovery
- Lessons learned

## Actions Taken
- Analyzed authentication logs
- Identified suspicious activity
- Classified incident severity
- Applied containment measures
- Addressed root cause
- Monitored system after recovery

## Preventive Measures
- Strong password policy
- Account lockout mechanisms
- Multi-Factor Authentication (MFA)
- Continuous log monitoring

## Deliverables
- Incident Response & Security Breach Simulation Report (PDF)
